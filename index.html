Inkrivel

Inkrivel is a paint shooter game, on which teams compete to paint the most of the map. It is online multiplayer, made with a completely custom engine, which uses OpenGL for graphics rendering and Linux syscalls to communicate between client and server.

Buzzfall

Buzzfall is a NES (Nintendo Entertainment System) game made in pure 6502 assembly. It is a simple Towerfall clone where two players kill each other with a limited supply of arrows. They can wall-jump and grab arrows stuck on the walls or on the ground.

Medieval Defense

Medieval Defense is a competitive tower-defense game, in which you can spend money to buy new towers to protect yourself or buy monsters that go attack your enemy. The game was made for the Nintendo DS so that each screen can show one player's battlefield. It also comes with a server to enable online play.

Hexis

Hexis is a tetris clone for the Game Boy Advance. It has most features expected from such a game, like hold and next pieces. Also, it was made completely in ARM assembly and uses the console hardware features directly.

Lumos

Lumos is a Game Boy Advance created at the Meeting Game Jam. It is a 2D adventure game in which you charge up your mana by sucking the flame out of candles and then use magic spells to kill monsters.

IronGB

IronGB is an original GameBoy emulator. It emulates the CPU (an 8 bit processor) with an interpreter approach, while also emulating the PPU (the chip that deals with drawing to the screen) all through software.

N.E.S. - Nosso Emulador Supimpa

N.E.S. is a Nintendo Entertainment System emulator, which emulates the CPU with an interpreter and the PPU via software only. It has pretty good compatibility, although limited in scope by not implementing mappers, which were common on NES cartridges.
