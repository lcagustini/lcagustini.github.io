N.E.S.

Personal work

N.E.S. is a Nintendo Entertainment System emulator, which emulates the CPU with an interpreter and the PPU (graphics) via software only

It has pretty good compatibility, although limited in scope by not implementing mappers, which were common on NES cartridges

Iron GB

Personal work

IronGB is an original GameBoy emulator. It emulates the CPU (an 8 bit processor) with an interpreter, while also emulating the PPU (graphics) all through software

It implements a debugger that allows the user to step through each instruction
