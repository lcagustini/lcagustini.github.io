Medieval Defense Mapper

A tool to help create collision data for the game Medieval Defense
It shows the game map with an overlay that the designer can click to define what type of collision should each tile have
The tool then prints ou the collision data that can be copied over to the game and used at runtime

GBA Tile Viewer

GBA image and tile data is store as an assembly file, this tool helps visualize the raw tile data
Helped with the development of Hexis

Img to CHR

A command line tool to convert images to a NES compatible CHR binary format
This was used to develop Buzzfall
