Spacelines from the Far Out

Spacelines from the Far Out is a roguelite co-op management game about airlines in space, featuring unlockable spaceships, upgrades, characters and customizations, randomly-generated challenges and trips.

The game features online and local multiplayer and is available on Steam and Xbox Game Pass.

Project GS

A raycast engine similar to the technique used by Wolfenstein 3D, but with floor and ceiling textures

This project was also a playground for threading, by dividing each frame in columns that separate threads can work on

Medieval Defense

Medieval Defense is a competitive tower-defense game, in which you can spend money to buy new towers to protect yourself or buy monsters that go attack your enemy.

The game was made for the Nintendo DS so that each screen can show one player's battlefield. It also comes with a server to enable online play.

PCTan

A clone of the BBTan mobile game made in Rust and SDL 2

Lumos

Lumos is a Game Boy Advance created at the Meeting Game Jam in 24h

It is a 2D adventure game in which you charge up your mana by absorbing the flame out of candles and then using magic spells to kill monsters.

Hexis

Hexis is a Tetris clone for the Game Boy Advance.

It has most features expected from such a game, like hold and next pieces. Also, it was made completely in ARM assembly and uses the console hardware features directly.

Buzzfall

Buzzfall is a NES (Nintendo Entertainment System) game made in pure 6502 assembly in a week-long game jam

It is a simple Towerfall clone where two players kill each other with a limited supply of arrows. They can wall-jump and grab arrows stuck on the walls or on the ground
